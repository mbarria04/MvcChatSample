# Procedimiento: Gestión de Roles y Seguridad

## Ubicación en Menú
Para gestionar esta sección, el usuario debe navegar en la barra lateral:
1. **Administración** (Menú Principal)
2. **Roles** (Submenú)

## Paso 1: Crear un Rol
1. Dentro de la pantalla 'Roles', pulsar el botón **Nuevo Rol**.
2. Campos a completar:
   - **Nombre**: (Ej: Administrador, Supervisor)
   - **Descripción**: Resumen de funciones.
   - **Estado**: Activo/Inactivo.
3. Guardar cambios.

## Paso 2: Configurar Permisos
- **Menús Permitidos**: Seleccionar acceso a Ventas, Compras o Reportes.
- **Acciones Permitidas**: Marcar Crear, Editar, Eliminar o Consultar.

## Paso 3: Asignación a Usuario
1. Navegar a: **Administración** -> **Usuarios**.
2. Editar el perfil del usuario deseado.
3. En el campo 'Rol', seleccionar el nombre del rol creado anteriormente.
4. Guardar.

## Reglas de Negocio
- Requiere permiso de **Administrador**.
- Un rol con usuarios asignados no puede ser eliminado.
